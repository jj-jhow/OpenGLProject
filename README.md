# OpenGLProject
OpenGL C++ game

## Dependencies
- bullet - for physics engine
- freetype - for ui font
- glew-2.1.0
- glfw-3.3.8.bin.WIN64
- glm
- stb-master

## How to play
- 'Enter' to start the game
- 'Up' or 'W' to jump
